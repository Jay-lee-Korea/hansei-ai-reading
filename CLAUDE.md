# CLAUDE.md (V2 — 토큰 절감형)

한세대학교 도서관 AI 리터러시 프로그램 — "AI 독서 파트너와 고전 읽기"

> **V2의 목적**: 학생들이 Claude Pro 구독으로 수업을 듣는다. 5시간 토큰 윈도우 안에 한 회차를 끝내야 한다. 이 버전은 v1 대비 **토큰 소비를 60% 이상 줄인** 설계이다.

## 모델 가이드 (필수)

수업 시작 시 학생은 반드시 `/model` 명령으로 **Haiku 4.5**로 전환한다.

| 상황 | 권장 모델 | 이유 |
|------|---------|------|
| 기본 (모든 안내·퀴즈·짧은 분석) | `claude-haiku-4-5` | 빠르고 토큰 적게 씀 |
| 에세이 초안 생성 (`/에세이코치` Block 3) | Sonnet (1회만) | 글의 구조·문체 품질 |
| 에세이 심사 (`/심사위원`) | Sonnet (1회만) | 평가 일관성 |

> **원칙**: 의심되면 Haiku. Sonnet은 "한 번에 결과물이 나오는 무거운 생성"에만.

## 토큰 절감 규칙

1. **WebSearch / WebFetch 사용 안 함** — `.claude/settings.local.json`에서 deny 처리됨
2. **각 응답은 캡(상한)을 지킴** — 각 SKILL.md 상단의 "응답 길이 캡" 표 참고
3. **Phase A는 짧게, Phase B는 더 짧게** — 불필요한 반복·요약 금지
4. **사실 검증은 AI가 아닌 학생이 직접 수행** — AI는 검증 가이드만 제공

## 구조

- `.claude/skills/` — 6개 학습 스킬 (슬래시 커맨드로 수업 진행)
- 각 스킬은 `SKILL.md` + `references/` (블록별 교안) 구조
- `고전텍스트/` — 실습용 고전 발췌본
- `자료/` — 강사·학생용 보조 문서
