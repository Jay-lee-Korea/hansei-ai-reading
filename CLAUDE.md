# CLAUDE.md (V2 — 토큰 절감형)

한세대학교 도서관 AI 리터러시 프로그램 — "AI 독서 파트너와 고전 읽기"

> **V2의 목적**: 학생들이 Claude Pro 구독으로 수업을 듣는다. 5시간 토큰 윈도우 안에 한 회차를 끝내야 한다. 이 버전은 v1 대비 **토큰 소비를 60% 이상 줄인** 설계이다.

## 모델 가이드 (필수)

수업 시작 시 학생은 반드시 `/model` 명령으로 **Haiku 4.5**로 전환한다.

| 상황 | 권장 모델 | 이유 |
|------|---------|------|
| 기본 (모든 안내·퀴즈·짧은 분석) | `claude-haiku-4-5` | 빠르고 토큰 적게 씀 |
| 에세이 초안 생성 (`/에세이코치` Block 3) | Sonnet (1회만) | 글의 구조·문체 품질 |
| 에세이 심사 (`/심사위원`) | Sonnet (1회만) | 평가 일관성 |

> **원칙**: 의심되면 Haiku. Sonnet은 "한 번에 결과물이 나오는 무거운 생성"에만.

## 토큰 절감 규칙

1. **WebSearch / WebFetch 사용 안 함** — `.claude/settings.local.json`에서 deny 처리됨
2. **각 응답은 캡(상한)을 지킴** — 각 SKILL.md 상단의 "응답 길이 캡" 표 참고
3. **Phase A는 짧게, Phase B는 더 짧게** — 불필요한 반복·요약 금지
4. **사실 검증은 AI가 아닌 학생이 직접 수행** — AI는 검증 가이드만 제공

## 구조

- `.claude/skills/` — 6개 학습 스킬 (슬래시 커맨드로 수업 진행)
- 각 스킬은 `SKILL.md` + `references/` (블록별 교안) 구조
- `고전텍스트/` — 실습용 고전 발췌본
- `자료/` — 강사·학생용 보조 문서
- `중간과제/` — 학생이 1회차와 2회차 사이 2주간 채우는 3개 템플릿 파일 (`탐구일지.md`, `prompts.md`, `에세이재료.md`). 2회차 `/에세이코치`가 이 폴더를 자동으로 Read해 개인화된 진행을 함

## 학생의 고전 텍스트 — 한 번만 선택 (필수)

학생은 1회차 시작 시 5편 중 한 편을 선택하고, 그 선택은 `자료/내선택.md`에 기록된다. 선택은 1·2회차 전 과정에서 **한 번만** 이뤄진다.

**모든 스킬이 따라야 할 규칙:**

1. 스킬 시작 시 `자료/내선택.md` 파일을 먼저 읽어 학생이 고른 텍스트 파일명을 확인한다.
2. 파일명이 `[미선택]` 상태이면:
   - 학생에게 `자료/고전텍스트선택안내.md`(5편 메뉴)를 보여주고 한 번만 선택을 받는다
   - 받은 답으로 `자료/내선택.md`의 `[미선택]`을 해당 파일명으로 교체한다 (Edit 도구 사용)
3. 파일명이 이미 정해져 있으면 **다시 묻지 않고** 그 파일을 자동으로 사용한다.
4. 각 스킬의 references에서 `[파일명]`이라고 표기된 자리는 모두 위 선택값으로 자동 치환한다.

> **첫 스킬 책임**: 1회차 첫 진입은 `/프롬프트도장`이다. 이 스킬이 시작 시점에 위 절차를 책임지고 진행한다. 다른 스킬은 이미 채워진 `자료/내선택.md`를 읽기만 하면 된다.

## 소통 규칙

- 한국어로 소통
- 대학생 눈높이, 친근하고 격려하는 톤
- 답을 바로 주기보다 스스로 생각하도록 유도
- **응답 길이는 짧고 명확하게** — 토큰을 아끼는 것이 곧 학생을 살리는 길
