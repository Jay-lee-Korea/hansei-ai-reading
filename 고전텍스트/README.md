# 고전텍스트 폴더

한세대학교 도서관 AI 리터러시 프로그램 — "AI 독서 파트너와 고전 읽기"의 실습용 발췌본 모음.

## 수록 텍스트 (5편)

| 파일 | 작품 | 저자 | 분량 |
|---|---|---|---|
| `동굴의비유.txt` | 『국가』 제7권 — 동굴의 비유 | 플라톤 | 약 50줄 |
| `논어_학이편.txt` | 『논어』 제1편 — 학이 | 공자 (제자 편집) | 약 70줄 |
| `니코마코스윤리학_제2권.txt` | 『니코마코스 윤리학』 제2권 | 아리스토텔레스 | 약 70줄 |
| `도덕경.txt` | 『도덕경』 발췌 (1·2·8·11·17·48장) | 노자 | 약 80줄 |
| `성찰_제1_제2.txt` | 『제일철학에 관한 성찰』 1·2성찰 | 데카르트 | 약 80줄 |

각 파일은 동일한 6단 포맷으로 구성된다 (V2 — 2026.5.7 리팩토링):
1. **[강사 메모]** — 권장 읽기 시간, 난이도(★~★★★★★), 걸려 넘어지기 쉬운 개념, 보조 자료 권장 여부
2. **[배경]** — 작품 출처, 시대, 표준판 정보
3. **[직접 인용 — 원문]** ⭐ — 한문/그리스어/라틴어 원문 + 강사 직역. `/팩트체커`의 검증 기준점
4. **[발췌 요약 — 강사 재구성]** — 강의용 요약. 원문이 아니라 해석본임을 명시
5. **[핵심 개념]** — 핵심 용어 4~6개
6. **[보편 논점]** + **[연결 시도 — 선택적]** — 시대 무관한 철학적 물음 5개와, 토론 후반부에서만 사용할 AI 시대 연결 질문 3개로 분리

> ⭐ V1과의 차이: V1은 "AI 시대 토론질문"이 텍스트 끝에 일괄 노출되어 학생 사고를 미리 가뒀다. V2는 보편 논점을 먼저 다루고 AI 연결은 선택적 후반부 활동으로 분리했다. 또한 V1에는 직접 인용이 거의 없어 `/팩트체커`가 깊이 작동하지 못했다 — V2에서 원전 직접 인용 박스를 신설했다.

## 출처와 번역

이 폴더의 텍스트는 **원전을 직접 복제한 것이 아니라**, 강의 목적에 맞춰 **강사(이승호)가 발췌·요약·재구성한 학습용 자료**이다. 한문 원문 인용은 정본을 따랐고, 번역은 강사가 다듬었다. 따라서 학술 인용에는 적합하지 않다.

원전 전문이 필요한 학생은 다음 공개 소스를 참조하면 된다.

### 한국어 원전 / 번역
- **위키문헌(ko)** — https://ko.wikisource.org/ — CC BY-SA 4.0
- **동양고전종합DB** — https://db.cyberseodang.or.kr/ — 교육용 무료
- **한국고전번역원 DB** — https://db.itkc.or.kr/ — 한국 한문고전 번역
- **국립중앙도서관 LOD/OpenAPI** — https://www.nl.go.kr/

### 영문 원전
- **Project Gutenberg** — https://www.gutenberg.org/ — 퍼블릭 도메인
  - 플라톤 『국가』 #1497, 아리스토텔레스 『니코마코스 윤리학』 #8438
  - 데카르트 『성찰』 #59, 노자 『도덕경』 (J. Legge 영역) #216
- **Standard Ebooks** — https://standardebooks.org/ — CC0
- **Perseus Digital Library** — http://www.perseus.tufts.edu/ — 그리스·라틴어 원전

### 자동 다운로드 예시
```bash
# 플라톤 국가 영문 전문
curl -s https://www.gutenberg.org/files/1497/1497-0.txt > republic_en.txt

# 위키문헌 논어 전문 (REST API)
curl -s "https://ko.wikisource.org/w/api.php?action=query&titles=논어&format=json&prop=extracts&explaintext=true"
```

## 라이선스

- 이 폴더의 발췌 요약본은 **수업 내부용**으로만 사용한다.
- 외부 배포 시에는 강사 표기 + "한세대학교 강의용 발췌본" 명기를 권장한다.
- 인용된 한문 원문 자체는 **퍼블릭 도메인**(저자 사망 2,000년 이상 경과).

## 텍스트 추가/교체 가이드

다른 고전을 추가하고 싶다면 동일 포맷을 유지하라:

```
[저자, 작품, 부분] (수업용 발췌본)
━━━
[강사 메모]
- 권장 읽기 시간: ○분
- 난이도: ★★★ (5점 만점)
- 걸려 넘어지기 쉬운 개념: ⋯
- 보조 자료 권장: ⋯
━━━
[배경]  ← 100~150자, 표준판 정보 포함
━━━
[직접 인용 — 원문]
▣ 인용 1 (출처)
원문: ⋯ (한문/그리스어/라틴어/영어)
직역: ⋯ (강사 번역)
▣ 인용 2 ⋯ (3~5개)
━━━
[발췌 요약 — 강사 재구성]
> 직접 인용 외에는 강사의 해석을 거쳤음을 명시
⋯ 본문 ⋯
━━━
[핵심 개념] ← 4~6개 용어
[보편 논점] ← 시대 무관한 5개
━━━
[연결 시도 — 선택적]
> 보편 논점을 충분히 다룬 후에만 사용
⋯ AI 시대 연결 질문 3개 ⋯
```

이 포맷은 의도적으로 명시적이다:
- "원문 인용"과 "강사 재구성"을 시각적으로 분리해 학생이 무엇이 원전이고 무엇이 해석인지 알 수 있다
- AI 연결 질문을 후반부 선택 활동으로 분리해 학생의 사고가 처음부터 AI에 묶이지 않도록 한다
- 토큰 절감(Haiku 4.5) + 학생의 1회독 부담(15~25분) 모두를 만족한다
