# AI 독서 파트너와 고전 읽기 (V2 — 토큰 절감형)

한세대학교 도서관 AI 리터러시 프로그램 — 클로드 코드 실습 플러그인

## 이 프로그램은?

AI를 "브라우저 채팅창" 밖으로 꺼내, **터미널에서 고전을 함께 읽는** 경험을 합니다. 6가지 AI 학습 파트너(스킬)가 여러분의 비판적 사고를 훈련합니다.

> **V2의 차이점**: Claude Pro 사용자가 5시간 토큰 윈도우 안에 한 회차를 안전하게 완주할 수 있도록 최적화된 버전입니다. 웹 리서치는 학생이 직접 수행하고, 기본 모델은 Haiku 4.5입니다.

## 빠른 시작

### 1. 사전 준비

- [Node.js](https://nodejs.org) LTS 설치
- 클로드 코드 설치: `npm install -g @anthropic-ai/claude-code`
- Claude Pro 계정 로그인

### 1-1. 수업 시작 직전 — 모델을 Haiku로! ⭐

클로드 코드 실행 직후 아래 명령어를 입력하세요:

```
/model
```

목록에서 **Haiku 4.5** 를 선택합니다. 이 한 단계가 토큰 소비를 가장 크게 줄입니다.
(에세이 초안 / 심사 단계에서만 강사 안내에 따라 Sonnet으로 잠시 전환합니다.)

### 2. 이 레포 다운로드

```bash
cd ~/Desktop
git clone https://github.com/Jay-lee-Korea/hansei-ai-reading.git
cd hansei-ai-reading
```

### 3. 클로드 코드 실행

```bash
claude
```

### 4. 첫 스킬 — `/프롬프트도장`이 고전 텍스트 선택을 안내합니다

수업 첫 진입 시 `/프롬프트도장`이 5편의 고전 메뉴(`자료/고전텍스트선택안내.md`)를 보여주고 한 번만 선택을 받습니다. 한 번 고른 텍스트는 `자료/내선택.md`에 기록되어 1·2회차 모든 스킬에서 자동으로 사용됩니다 — 다시 묻지 않습니다.

### 5. 스킬 사용

아래 명령어를 입력하면 각 학습 파트너가 활성화됩니다:

| 명령어 | 학습 파트너 | 하는 일 |
|--------|-----------|--------|
| `/소크라테스` | 산파술 대화 | 질문으로만 대화하며 사고를 깊게 만듦 |
| `/분석마스터` | 고전 분석가 | 텍스트 파일을 읽고 체계적으로 분석 |
| `/프롬프트도장` | 프롬프트 사범 | AI에게 잘 질문하는 법을 Before/After로 훈련 |
| `/에세이코치` | 에세이 편집장 | 5단계로 비판적 에세이 작성을 안내 |
| `/심사위원` | 학술 심사관 | 에세이를 4개 기준으로 평가하고 피드백 |
| `/팩트체커` | 검증 탐정 | AI가 생성한 텍스트의 사실 여부를 검증 |

## 폴더 구조

```
.
├── .claude/skills/     ← 6개 AI 학습 파트너 (스킬)
├── CLAUDE.md           ← 프로젝트 설정
├── 고전텍스트/          ← 실습용 고전 발췌본
└── 자료/               ← 학생용 보조 자료 (사전설치, 과제 안내서, 프롬프트 모음 등)
```

## 고전 텍스트 추가하기

`고전텍스트/` 폴더에 `.txt` 파일을 넣으면 자동으로 인식됩니다.

```bash
# 예시: 새 텍스트 추가
cp 나의텍스트.txt 고전텍스트/
```

## 시상 및 지원

- **우수 에세이 시상** (총 100만원 규모, 우수작 3편 내외): 최우수 50 / 우수 30 / 장려 20만원
- **모든 참가자**: Claude Pro 1개월, 도서관 굿즈 및 포인트 장학금 1포인트, 도서관장 명의 수료증
- 자세한 평가 기준·등급표·시상 절차는 `자료/중간과제안내서.md` 참고

## 토큰을 아끼는 4가지 습관

1. 수업 시작 시 `/model`로 **Haiku 4.5** 선택
2. 한 번에 긴 답을 요구하지 말고, **짧게 여러 번** 묻기
3. AI에게 "웹에서 검색해줘" 하지 않기 — 검색은 **본인의 브라우저에서**
4. 만족스러운 답이 나오면 **거기서 멈추기** (불필요한 후속 요청 자제)
