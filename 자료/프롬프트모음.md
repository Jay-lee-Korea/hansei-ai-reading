## A. 텍스트 요약 & 구조 파악 (기초)

### A1. 핵심 요약
```
이 텍스트 파일을 읽고 핵심 내용을 3문장으로 요약해줘.
```

### A2. 단계별 논증 정리
```
이 텍스트의 저자가 주장하는 핵심 논증을 단계별로 정리해줘.
전제 → 근거 → 결론 구조로 보여줘.
```

### A3. 핵심 개념 정리
```
이 텍스트에서 가장 중요한 개념 5개를 뽑고,
각각을 대학생이 이해할 수 있는 쉬운 말로 설명해줘.
```

### A4. 목차 재구성
```
이 텍스트의 내용을 바탕으로, 만약 내가 이 주제로 발표를 한다면
어떤 목차로 구성하면 좋을지 제안해줘.
```

## B. 비판적 분석 (중급)

### B5. 논증의 약점 찾기
```
이 텍스트의 주장에서 논리적으로 약한 부분을 3가지 찾아줘.
각각에 대해 왜 약한지 설명하고, 반론을 제시해줘.
```

### B6. 전제 의심하기
```
이 텍스트의 저자가 당연하다고 가정하고 있는 전제(숨겨진 가정)는 무엇인가?
그 전제가 틀릴 수 있는 경우를 제시해줘.
```

### B7. AI vs 인간 비교
```
같은 폴더에 있는 ai_essay.txt와 human_essay.txt를 둘 다 읽고,
논증 구조, 문체, 깊이, 독창성 측면에서 비교 분석해줘.
어떤 글이 AI가 쓴 것 같고, 왜 그렇게 판단했는지 설명해줘.
```

### B8. 편향 탐지
```
이 텍스트에서 저자의 편향이나 특정 관점에 치우친 부분이 있는지 분석해줘.
어떤 관점이 누락되었는지도 알려줘.
```

## C. 관점 전환 (중급)

### C9. 특정 철학자의 눈으로
```
만약 니체가 이 텍스트를 읽는다면 어떤 반응을 보일까?
니체의 철학적 관점에서 이 텍스트를 비평해줘.
```

### C10. 반대 입장 구축
```
이 텍스트의 핵심 주장에 대해 정반대 입장을 가진 사람의 논증을 만들어줘.
가능한 한 설득력 있게 작성해줘.
```

### C11. 시대 전환
```
이 텍스트가 쓰인 시대적 맥락을 설명하고,
만약 저자가 2026년 한국에 살고 있다면 같은 주장을 어떻게 수정했을지 상상해줘.
```

### C12. 다학제적 분석
```
이 텍스트의 주장을 경제학, 심리학, 사회학 관점에서 각각 분석해줘.
어떤 학문 분야에서 이 주장이 가장 강하고, 어디서 가장 약한가?
```

## D. 검증 가이드 설계 (V2 — 학생 주도 리서치)

> **V2 변경**: 웹 리서치는 AI에 맡기지 않습니다. AI는 **검색·검증의 가이드**만 만들고, 실제 검색은 본인의 브라우저에서 직접 합니다. 이것이 AI 리터러시의 본질이고, 동시에 토큰을 가장 크게 절약하는 방법입니다.

### D13. 검색어 후보 설계
```
이 텍스트의 핵심 주장이 현대 사회에 적용되는 사례를 내가 직접 구글에서 찾아볼 거야.
검색어 후보 5개와, 각 검색어가 어떤 결과를 노리는지 한 줄로 알려줘.
웹 리서치는 하지 마.
```

### D14. AI 시대와의 연결 (사고만, 검색 X)
```
이 고전 텍스트의 주장을 AI/생성형 AI 시대에 적용하면 어떤 시사점이 있는지 분석해줘.
저자가 ChatGPT, 클로드 같은 AI를 봤다면 뭐라고 했을까?
(웹 검색은 하지 말고, 텍스트와 일반 지식으로만 답해줘)
```

### D15. 학술 검색 가이드
```
이 텍스트에 대한 학술적 해석을 내가 직접 찾아보고 싶어.
1. 추천 학술 데이터베이스 3곳 (KCI, Google Scholar 등)과 각각의 검색어
2. 살펴볼 만한 한국어 키워드 5개
3. 어떤 학자가 이 분야 권위자로 알려져 있는지(일반 지식 범위)
이걸 정리해줘. 직접 검색하지는 마.
```

### D16. 팩트체크 계획서 (검증은 본인 몫)
```
이 텍스트에서 사실(fact)에 해당하는 주장 5개를 뽑고,
각 주장에 대해:
- 검증할 검색어 후보 3개
- 신뢰할 만한 출처 유형(.gov, .edu, 학술DB, 1차 사료 등)
- 가장 의심스러운 부분 한 줄
을 표로 정리해줘. 검증은 내가 직접 브라우저에서 할 거야.
```

## E. 에세이 & 글쓰기 (심화)

### E17. 에세이 구조 설계
```
"[고전 텍스트 제목]의 [핵심 주장]은 현대 사회에서 여전히 유효한가?"
라는 주제로 800~1200자 에세이의 구조를 설계해줘.
서론-본론-결론 구조와 각 단락의 핵심 논점을 제시해줘.
```

### E18. 비판적 에세이 초안
```
아래 구조에 맞춰 비판적 에세이 초안을 작성해줘.
단, 저자의 주장을 단순히 요약하지 말고,
나(학생)의 비판적 관점이 드러나도록 써줘.

[에세이 구조를 여기에 붙여넣기]
```

### E19. 초안 비판적 검토
```
내가 작성한 에세이를 읽고 아래 기준으로 피드백해줘:
1. 논증의 논리적 일관성
2. 근거의 충분성
3. 반론에 대한 고려
4. 문체와 가독성
5. AI에 의존한 부분이 있는지 (독창성)

essay.md 파일을 읽어줘.
```

### E20. 소크라테스식 대화
```
지금부터 소크라테스처럼 나에게 질문해줘.
주제: [고전 텍스트의 핵심 주제]
내가 대답하면, 그 대답의 모순이나 불완전한 점을 짚어서 다시 질문해줘.
내가 "그만"이라고 할 때까지 계속해줘.
```

## 프롬프트 활용 팁

1. **구체적으로 요청하세요**: "분석해줘" 보다 "논증 구조를 3단계로 나눠서 분석해줘"가 낫습니다
2. **역할을 부여하세요**: "니체의 관점에서", "경제학자로서" 등 관점을 지정하면 더 깊은 답변을 얻습니다
3. **후속 질문하세요**: 첫 답변이 마음에 안 들면 "좀 더 구체적으로", "반대 관점도 포함해서" 등으로 이어가세요
4. **비판적으로 읽으세요**: AI의 답변을 그대로 받아들이지 말고, "이건 맞아?", "출처가 있어?" 라고 확인하세요
5. **자신만의 프롬프트를 만들어보세요**: 위 예시를 변형해서 나만의 질문을 만드는 것이 진짜 실력입니다
