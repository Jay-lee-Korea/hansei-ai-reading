# 클로드 코드 사전 설치 가이드

> 한세대학교 도서관 AI 리터러시 프로그램 "AI 독서 파트너와 고전 읽기"
> 강의 전 반드시 아래 단계를 완료해주세요. 예상 소요시간: 20~30분

---

## 준비: 터미널 열기

설치 작업은 모두 **터미널**(검은 화면에 명령어를 입력하는 프로그램)에서 진행합니다. 먼저 터미널 여는 법부터 익혀주세요.

### Mac에서 터미널 열기

**방법 1 (가장 빠름)**: Spotlight 검색
1. `Command(⌘) + Space` 키를 동시에 누름
2. 검색창에 `터미널` 입력 (또는 `Terminal`)
3. `Enter` 키 → 검은 창이 뜸

**방법 2**: Finder에서 직접 찾기
- `응용 프로그램` 폴더 → `유틸리티` 폴더 → `터미널.app` 더블클릭

> **팁**: Dock(하단 바)에 자주 쓰도록 추가하려면, 터미널이 켜져 있는 상태에서 Dock 아이콘 우클릭 → "옵션" → "Dock에 유지" 선택

### Windows에서 터미널 열기

Windows 11 기준 **PowerShell**(또는 Windows Terminal) 사용을 권장합니다.

1. 시작 버튼(⊞) 클릭 또는 `Windows` 키 누름
2. `PowerShell` 입력
3. `Windows PowerShell` 클릭 → 파란/검은 창이 뜸

> **권한 오류가 날 때만**: 검색 결과에서 우클릭 → "관리자 권한으로 실행"

## 1단계: Node.js 설치

클로드 코드는 Node.js 위에서 동작합니다.

### Windows

1. https://nodejs.org 접속
2. **LTS** (장기 지원 버전) 다운로드 클릭
3. 다운로드된 `.msi` 파일 실행
4. 설치 마법사에서 **모두 "Next"** 클릭 (기본 설정 유지)
5. 설치 완료

### Mac

1. https://nodejs.org 접속
2. **LTS** (장기 지원 버전) 다운로드 클릭 — Apple Silicon(M1/M2/M3/M4)이든 Intel이든 동일한 파일을 받으면 됩니다
3. 다운로드된 `.pkg` 파일을 더블클릭하여 실행
4. 설치 마법사에서 **모두 "계속"** → "동의" → "설치" 클릭
5. Mac 로그인 비밀번호 입력 (관리자 인증)
6. "설치 완료" 메시지가 뜨면 닫기

#### ⚠️ "확인되지 않은 개발자" 경고가 뜰 때

최신 macOS에서 가끔 **"확인할 수 없는 개발자가 배포한 항목이므로 열 수 없습니다"** 경고가 뜰 수 있습니다.

1. 경고창은 일단 **"확인"**으로 닫기
2. **시스템 설정** 열기 (Apple 메뉴 → 시스템 설정)
3. 좌측 메뉴에서 **개인정보 보호 및 보안** 클릭
4. 아래로 스크롤하면 차단된 항목 안내가 보임 → **"그래도 열기"** 클릭
5. 다시 `.pkg` 파일을 더블클릭하면 정상 설치됨

> nodejs.org의 공식 설치 파일은 안전합니다. 이 경고는 macOS 보안 정책 때문에 가끔 뜨는 것입니다.

### 설치 확인

터미널을 열고 아래 명령어를 입력합니다:

```
node --version
```

`v22.x.x` 같은 버전 번호가 출력되면 성공입니다.

## 2단계: Git 설치 확인

실습 자료를 GitHub에서 다운로드하기 위해 Git이 필요합니다.

### 확인

```
git --version
```

버전 번호가 출력되면 이미 설치되어 있습니다. 출력이 안 되면 아래를 따라하세요:

### Windows
1. https://git-scm.com 접속
2. "Download for Windows" 클릭
3. 설치 마법사에서 모두 "Next" → 설치 완료
4. **터미널을 완전히 닫고 다시 열기**

### Mac

Mac은 별도 다운로드 없이 터미널에서 자동 설치됩니다.

1. 터미널을 열고 다음을 입력:
   ```
   git --version
   ```
2. 설치되어 있지 않으면 **"command line developer tools를 설치하려면..."** 팝업이 뜸
3. **"설치"** 버튼 클릭 (Get Xcode 아니라 "설치"입니다)
4. 사용 약관 → **"동의"** 클릭
5. **5~10분 기다리기** (네트워크 속도에 따라 다름. 진행 막대가 나타남)
6. "소프트웨어가 설치되었습니다" 메시지가 뜨면 **"완료"** 클릭
7. 터미널에서 다시 `git --version` 입력 → 버전 번호가 출력되면 성공

> Xcode 전체가 아닌 **Command Line Tools만 설치**되는 것이라 용량 부담은 적습니다(약 1~2GB).

## 3단계: 클로드 코드 설치

터미널에서 아래 명령어를 **복사해서 붙여넣기** 하세요:

```
npm install -g @anthropic-ai/claude-code
```

설치가 진행되며, 완료까지 1~2분 소요됩니다.

### 설치 확인

```
claude --version
```

버전 번호가 출력되면 성공입니다.

## 4단계: 실습 자료 다운로드 (GitHub 클론)

바탕화면에 실습 폴더를 다운로드합니다:

```
cd ~/Desktop
git clone https://github.com/[레포주소]/hansei-ai-reading.git
cd hansei-ai-reading
```

> Windows에서 `~/Desktop`이 안 되면: `cd %USERPROFILE%\Desktop` 을 사용하세요

바탕화면에 `hansei-ai-reading` 폴더가 생성됩니다.

### Git을 못 쓰는 경우

GitHub 페이지에서 **"Code" → "Download ZIP"** 을 클릭하여 ZIP 파일을 다운로드하고, 바탕화면에 압축을 풀어도 됩니다.

## 5단계: 클로드 코드 첫 실행 & 로그인

```
cd ~/Desktop/hansei-ai-reading
claude
```

처음 실행하면 로그인 화면이 나타납니다:
1. **"Use Claude.ai account (OAuth)"** 선택
2. 브라우저가 열리면 Claude Pro 계정으로 로그인
3. "Authorize" 버튼 클릭
4. 터미널로 돌아오면 로그인 완료

## 6단계: 스킬 확인

로그인 후 아래를 입력해보세요:

```
어떤 스킬을 사용할 수 있어?
```

6개 스킬 목록이 나오면 모든 준비가 끝났습니다:
- `/소크라테스` — 산파술 대화
- `/분석마스터` — 고전 텍스트 분석
- `/프롬프트도장` — 프롬프트 훈련
- `/에세이코치` — 에세이 작성 코치
- `/심사위원` — 에세이 심사
- `/팩트체커` — AI 결과물 검증

**맛보기**: `/프롬프트도장` 을 입력하고 "1"을 선택해보세요!

종료하려면 `/exit` 을 입력하세요.

## 7단계 ⭐ (V2 필수): 모델을 Haiku 4.5로 전환

수업이 시작되면 **반드시 첫 단계로** 모델을 Haiku로 전환합니다.

```
/model
```

목록이 뜨면 **Haiku 4.5** 를 선택하세요.

### 왜 Haiku로 바꾸나요?

- Claude Pro는 5시간 토큰 윈도우가 있어, 토큰을 다 쓰면 수업이 멈춥니다
- Haiku 4.5는 안내·퀴즈·짧은 분석에 충분하고, **토큰 소비가 가장 적습니다**
- 에세이 초안 생성(`/에세이코치` Block 3), 심사위원 평가(`/심사위원`) 단계에서만 강사 안내에 따라 잠깐 Sonnet으로 전환합니다

### 토큰 절약 4계명

1. 시작할 때 `/model`로 **Haiku 4.5** 선택
2. AI에게 "웹에서 검색해줘"라고 시키지 않기 — 검증·검색은 본인 브라우저에서
3. 한 번에 긴 답을 요구하지 말고, **짧게 여러 번** 묻기
4. 만족스러운 답이 나오면 **거기서 멈추기**

## 터미널 기초 명령어 치트시트

| 명령어 | 뜻 | 예시 |
|--------|-----|------|
| `cd 폴더명` | 폴더로 이동 | `cd Desktop` |
| `cd ..` | 상위 폴더로 이동 | `cd ..` |
| `ls` (Mac) / `dir` (Win) | 현재 폴더의 파일 목록 | `ls` |
| `pwd` (Mac) / `cd` (Win) | 현재 위치 확인 | `pwd` |
| `clear` | 화면 지우기 | `clear` |

**팁**: 이 문서에서 **복사 → 붙여넣기** 하세요!
- Windows 터미널 붙여넣기: `Ctrl + V` 또는 `마우스 우클릭`
- Mac 터미널 붙여넣기: `Cmd + V`

## 트러블슈팅 FAQ

### Q. `node`를 찾을 수 없다고 나와요
→ 터미널을 **완전히 닫고 다시 열어보세요**. 설치 후 터미널을 재시작해야 PATH가 적용됩니다.

### Q. `npm install` 에서 권한 오류가 발생해요 (Windows)
→ 터미널을 **관리자 권한으로 실행**하세요:
시작 메뉴에서 "명령 프롬프트" 검색 → 우클릭 → "관리자 권한으로 실행"

### Q. `npm install` 에서 권한 오류가 발생해요 (Mac)
→ 명령어 앞에 `sudo`를 붙이세요:
```
sudo npm install -g @anthropic-ai/claude-code
```
Mac 로그인 비밀번호를 묻습니다. **비밀번호를 입력해도 화면에 아무 글자도 표시되지 않는데 정상입니다** (보안을 위한 macOS의 동작). 그냥 입력하고 `Enter`를 누르세요.

### Q. Mac에서 "Operation not permitted" 오류가 나요
→ 터미널에 디스크 접근 권한이 없을 수 있습니다.
**시스템 설정 → 개인정보 보호 및 보안 → 전체 디스크 접근 권한** 에서 터미널을 허용 목록에 추가하세요.

### Q. `git clone`이 안 돼요
→ Git이 설치되어 있는지 확인하세요. 안 되면 GitHub에서 ZIP 다운로드로 대체하세요.

### Q. 클로드 코드 로그인이 안 돼요
→ 브라우저에서 https://claude.ai 에 먼저 로그인이 되어 있는지 확인하세요.

### Q. 스킬이 안 보여요
→ `hansei-ai-reading` 폴더 안에서 `claude`를 실행했는지 확인하세요. 다른 폴더에서 실행하면 스킬이 로드되지 않습니다.

### Q. 그래도 안 되면?
→ 걱정하지 마세요! 강의 당일 첫 40분이 셋업 시간입니다. 강사가 직접 도와드립니다.
