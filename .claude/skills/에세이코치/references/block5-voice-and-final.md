# Block 5: 나의 관점 추가 + 최종 저장 (v1의 Block 5+6 통합)

## EXPLAIN

📝 **Step 5: AI가 절대 쓸 수 없는 것 — 당신만의 목소리, 그리고 마무리**

이 블록에서 두 가지를 한 번에 마칩니다:
1. **본인만의 관점**을 에세이에 직접 삽입
2. 최종 점검 후 `essay_final.md`로 저장

### AI가 쓸 수 없는 4가지

| 유형 | 예시 |
|------|------|
| 💭 개인적 감상 | "이 구절을 처음 읽었을 때 나는..." |
| 🗣️ 수업 경험 | "수업에서 토론할 때..." |
| 🌟 독창적 비유 | "이것은 마치 ~와 같다" (나만의 비유) |
| 🤔 한계 인정 | "이 에세이에서 다루지 못한 것은..." |

> 이 4가지 중 **최소 1가지**가 들어가면, 그 에세이는 어떤 AI도 똑같이 못 씁니다. 평가표의 "독창성(10점)"·"비판적 사고(40점)"에 직접 영향.

### 최종 점검 항목 (짧게)

```
□ 맞춤법/문법
□ 서론에서 던진 질문에 결론이 답하는가
□ 인용 출처 명시 (작품명·장/절)
□ 분량: 800~1200자 (v2 기준)
```

## EXECUTE

**1단계: 본인 문단을 직접 작성** (300자 이내, 손으로)

위 4가지 중 최소 1가지를 본인이 직접 씁니다. 완벽한 문장이 아니어도 됩니다 — 솔직한 거친 표현이 더 좋습니다.

**2단계: AI에게 삽입 + 최종 점검 요청**

```
아래 내용을 에세이의 적절한 위치에 삽입하고,
동시에 맞춤법·논리 흐름·인용 출처를 점검해줘.
내가 쓴 부분의 원래 의미는 절대 바꾸지 마.
최종본은 essay_final.md로 저장해줘.

[직접 쓴 문단을 여기에 붙여넣기]
```

**3단계: `essay_final.md`를 처음부터 끝까지 한 번 읽기**

이게 제출본입니다. AI가 다듬는 과정에서 본인 의미가 변형되지 않았는지 확인하세요.
