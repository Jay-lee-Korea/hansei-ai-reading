# Block 1: 논점 추출

> **이 블록의 작동 원칙**: SKILL.md 시작 단계에서 이미 `중간과제/에세이재료.md`를 Read 했다. 학생이 미리 정리한 **주제 후보 2~3개·잠정 입장·한 줄 가설**이 있으면 거기서 출발한다. 비어 있으면 아래 표준 흐름으로 진행한다.

## EXPLAIN

📝 **Step 1: 에세이의 출발점 — 논점을 찾아봅시다.**

에세이의 핵심은 "무엇에 대해 쓸 것인가"입니다. 고전 텍스트를 읽고 에세이 주제가 될 수 있는 **논점(thesis)**을 추출하는 것이 첫 번째 단계예요.

### 분기 — 학생 사전 작업 활용

**Case A: `중간과제/에세이재료.md`에 주제 후보가 채워져 있는 경우** (권장 흐름)
- 학생이 미리 정리한 후보 A/B/C와 잠정 입장을 **그대로 보여주며 출발**한다.
- AI에게 새로 논점을 추출시키지 않는다. 대신 학생이 정한 후보 중 하나를 골라 정교화하도록 돕는다.

**Case B: 비어 있거나 자리표시자만 있는 경우**
- 아래 표준 흐름(AI에게 논점 추출 요청)으로 진행한다.

### 좋은 논점의 조건

| 조건 | 설명 | 예시 |
|------|------|------|
| 📌 논쟁 가능 | 찬성과 반대가 모두 가능해야 한다 | "소크라테스의 죽음은 정당했는가?" |
| 🔍 구체적 | 텍스트의 특정 부분에 근거할 수 있어야 한다 | "동굴의 비유에서 '빛'은 진리인가 권력인가?" |
| 💡 현대적 연결 | 오늘날의 맥락과 연결할 수 있으면 더 좋다 | "플라톤의 이상국가론은 AI 거버넌스에 적용 가능한가?" |

### 논점 추출 방법

1. **핵심 주장 찾기** — 저자가 가장 강하게 주장하는 것은 무엇인가?
2. **반론 상상하기** — 이 주장에 반대하려면 어떤 근거가 필요한가?
3. **현대 연결** — 이 주장이 오늘날에는 어떤 의미를 가지는가?

AI에게 논점 추출을 요청하면, 혼자서는 떠올리지 못했을 관점도 발견할 수 있습니다. 단, **최종 선택은 반드시 본인**이 해야 합니다.

## EXECUTE

### Case A 흐름 (에세이재료.md에 주제 후보 있음 — 권장)

**1단계: 본인이 미리 적어둔 후보를 스스로 다시 읽기**

`중간과제/에세이재료.md`의 "1. 에세이 주제 후보" 섹션에 적힌 후보 A/B/C와 본인의 잠정 입장을 **소리 내어 한 번** 읽어보세요.

**2단계: 1순위로 정한 후보를 정교화**

이미 표시한 1순위 후보를 가지고 AI에게 정교화 요청을 합니다:

```
나는 [선택한 고전]에 대한 에세이를 쓰려고 한다.
내 잠정 논점은: "[에세이재료.md에서 옮겨 적은 1순위 입장]"
이 논점이
 (a) 너무 넓지는 않은지
 (b) 찬반 양쪽 근거가 모두 만들어질 수 있는지
 (c) 800~1200자 안에 다룰 수 있는 적정 범위인지
세 가지 관점에서 평가하고, 필요하면 좁히거나 좀 더 도발적인 방향으로 다듬는 제안을 해줘.
```

**3단계: 최종 논점을 한 문장으로 확정**

AI 피드백을 참고해 본인 논점을 한 문장으로 다시 적습니다. 필요하면 `중간과제/에세이재료.md`의 "6. 한 줄 가설" 섹션을 갱신하세요.

---

### Case B 흐름 (재료가 비어 있는 경우)

**1단계: AI에게 논점 추출 요청**

선택한 고전텍스트 파일을 기반으로, 아래와 같은 프롬프트를 입력하세요:

```
고전텍스트/[파일명]을 읽고 에세이 주제가 될 수 있는 논점 3가지를 추출해줘.
각 논점에 찬성 근거와 반대 근거를 정리해줘.
```

**2단계: 논점 하나를 선택하고 이유를 메모하세요**

AI가 제시한 3가지 논점 중 하나를 골라주세요. 고를 때 이 질문들을 스스로에게 해보세요:
- 이 논점에 대해 **내가 할 말**이 있는가?
- 찬반 어느 쪽이든 **근거를 댈 수 있는가?**
- 이 주제로 2000자를 **채울 수 있는가?**

선택한 논점과 선택 이유를 간단히 메모해두세요. 이것이 에세이의 방향이 됩니다.

## QUIZ

```json
AskUserQuestion({
  "questions": [{
    "question": "어떤 논점을 골랐나요? 그리고 왜 그 논점을 선택했나요?",
    "header": "📝 논점 확인"
  }]
})
```

학생의 답변을 확인한 후:
- 선택한 논점이 에세이 주제로 적합한지 피드백한다
- 논점이 너무 넓으면 좁히도록 안내한다
- 논점이 너무 좁으면 확장할 수 있는 방향을 제안한다
- "좋은 선택이에요! 이제 이 논점으로 구조를 잡아볼까요?"로 마무리한다
