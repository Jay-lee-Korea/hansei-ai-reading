# Block 2: 핵심 개념 — 개념 추출과 쉬운 설명

## EXPLAIN

📚 **핵심 개념 추출이란?**

고전 텍스트에는 **낯선 용어와 추상적 개념**이 가득합니다. 이 개념들을 현대적 언어로 번역하면 텍스트가 갑자기 이해됩니다.

### 추출 형식

핵심 개념을 아래 **표 형식**으로 정리합니다:

| # | 개념 | 원문 표현 | 쉬운 설명 | 현대적 비유 |
|---|------|-----------|-----------|-------------|
| 1 | 이데아 | "참된 실재" | 눈에 보이는 것 너머의 진짜 본질 | 소스 코드 vs 화면에 보이는 앱 |
| 2 | 동굴 | "어두운 곳" | 감각에만 의존하는 무지의 상태 | SNS 알고리즘 버블 |
| 3 | 그림자 | "벽에 비친 것" | 진짜라고 착각하는 가짜 정보 | 요약만 보고 원문은 안 읽기 |

### 좋은 "쉬운 설명"의 조건

1. **대학생이 바로 이해**할 수 있는 말
2. 원래 의미를 **왜곡하지 않는** 범위에서 단순화
3. **현대적 비유**로 친숙하게 연결

## EXECUTE

아래 프롬프트를 실행해보세요:

```
고전텍스트/[파일명]에서 핵심 개념 5개를 뽑아줘.
각 개념을 아래 표 형식으로 정리해줘:
| # | 개념 | 원문 표현 | 쉬운 설명 | 현대적 비유 |

대학생이 이해할 수 있는 말로 설명해줘.
```

결과를 읽어보고, "쉬운 설명"이 정말 쉬운지, "현대적 비유"가 적절한지 확인해보세요.
