# Block 1: 논증 분석 — 주장, 전제, 근거, 약점

## EXPLAIN

📚 **논증 구조 분석이란?**

모든 글에는 **논증 구조**가 숨어 있습니다. 논증 구조를 분석한다는 것은 글의 뼈대를 드러내는 작업입니다.

### 분석 순서

```
핵심 주장(Thesis) → 전제(Premises) → 근거(Evidence) → 숨겨진 가정 → 논리적 약점
```

| 단계 | 질문 | 설명 |
|------|------|------|
| 핵심 주장 | "저자가 궁극적으로 말하려는 것은?" | 글 전체를 한 문장으로 요약 |
| 전제 | "주장을 뒷받침하는 하위 주장은?" | 핵심 주장을 지탱하는 논거들 |
| 근거 | "전제를 뒷받침하는 증거는?" | 사실, 사례, 통계, 인용 등 |
| 숨겨진 가정 | "저자가 당연하게 여기는 것은?" | 명시하지 않았지만 전제하는 것 |
| 논리적 약점 | "어디가 공격 가능한가?" | 논리적 비약, 일반화, 순환논법 등 |

### 예시

> 플라톤의 동굴의 비유에서:
> - **핵심 주장**: 감각으로 인식하는 세계는 진짜가 아니다
> - **전제**: 동굴 밖 태양빛(이데아)이 더 실재적이다
> - **숨겨진 가정**: 이성적 인식이 감각적 인식보다 우월하다
> - **논리적 약점**: "동굴 밖"이 진짜라는 것은 어떻게 증명하는가?

```
2. 전제 3개를 찾아서 각각의 근거를 정리
3. 숨겨진 가정 2개를 찾아줘
4. 논리적 약점 2개를 지적해줘
```

> `[파일명]`은 학생이 1회차 시작 시 선택한 텍스트(`자료/내선택.md`)로 자동 치환됩니다. Claude가 이미 채워서 보여줍니다.
