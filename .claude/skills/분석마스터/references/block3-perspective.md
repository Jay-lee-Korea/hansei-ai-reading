# Block 3: 다관점 비평 — 다른 철학자의 눈으로

## EXPLAIN

📚 **다관점 비평이란?**

하나의 텍스트를 **다른 사상가의 관점**에서 바라보면 전혀 다른 모습이 보입니다. 마치 같은 건물을 앞에서, 옆에서, 위에서 보면 형태가 달라지는 것처럼요.

### 대표 관점

| 사상가 | 핵심 렌즈 | 이런 질문을 던진다 |
|--------|-----------|-------------------|
| 니체 | 권력 의지, 주인/노예 도덕 | "이 텍스트는 강자의 도덕인가, 약자의 도덕인가?" |
| 마르크스 | 계급, 생산관계, 이데올로기 | "이 사상은 누구의 이익을 대변하는가?" |
| 공자 | 인(仁), 예(禮), 관계 | "이 주장은 공동체의 조화에 기여하는가?" |
| 한나 아렌트 | 전체주의, 악의 평범성 | "이 논리가 극단으로 가면 어떤 결과를 낳는가?" |

### 비평 구조

```
1. [철학자]의 핵심 사상 요약 (2~3문장)
2. [철학자]가 이 텍스트를 읽었다면 동의할 부분
3. [철학자]가 이 텍스트를 읽었다면 비판할 부분
4. 비평에서 얻을 수 있는 새로운 통찰
```

> **참고**: 위 표의 4명 중 골라서 진행하세요. 웹 리서치는 사용하지 않습니다 (권한 차단됨). 학생 자신이 알고 있는 범위 + AI 학습 데이터로 충분히 깊은 비평이 가능합니다.

## EXECUTE

아래 순서로 실행해보세요:

**1. 관점 선택** — 니체, 마르크스, 공자, 한나 아렌트 중 하나를 선택하거나, 직접 다른 사상가를 지정하세요.

**2. 비평 실행** — 아래 프롬프트를 실행해보세요:

```
너는 [철학자]야.
고전텍스트/[파일명]을 [철학자]의 관점에서 비평해줘.
1. [철학자]의 핵심 사상을 2~3문장으로 요약
2. 이 텍스트에서 동의할 부분
3. 이 텍스트에서 비판할 부분
4. 이 비평에서 얻을 수 있는 새로운 통찰
```

결과를 읽어보고, 같은 텍스트가 다른 관점에서 어떻게 달라지는지 느껴보세요.
