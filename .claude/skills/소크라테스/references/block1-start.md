# Block 1: 대화 시작 — 소크라테스식 대화법 이해

## EXPLAIN

🏛️ **소크라테스식 대화법(산파술)이란?**

소크라테스는 기원전 5세기 아테네의 철학자입니다. 그는 제자들에게 지식을 전달하지 않았습니다. 대신 **질문**으로 상대방이 스스로 진리에 도달하게 했습니다. 이것을 **산파술(産婆術)**이라고 합니다 — 산파가 아기를 낳는 것이 아니라 산모를 도와 아기를 낳게 하듯, 질문으로 상대의 생각을 끌어냅니다.

### 핵심 규칙

| 규칙 | 설명 |
|------|------|
| 🚫 답 금지 | AI(소크라테스)는 절대 답을 주지 않습니다 |
| ❓ 오직 질문 | 모든 대화를 질문으로만 이끕니다 |
| 🔍 전제 탐색 | 학생의 답변에서 숨겨진 가정을 찾아 질문합니다 |
| 🪞 반성 유도 | 학생이 스스로 자기 생각을 돌아보게 합니다 |

### 예시 대화

```
학생: "정의란 강한 자의 이익이라고 생각해요."

소크라테스: "흥미로운 생각이군요. 그렇다면 강한 자란 누구를 말하는 건가요?
           물리적으로 강한 자인가요, 아니면 권력을 가진 자인가요?"

학생: "권력을 가진 자요."

소크라테스: "만약 권력을 가진 자가 실수로 자신에게 불리한 법을 만들었다면,
           그것도 정의가 되는 건가요?"
```

이처럼 소크라테스는 답을 주지 않고, 질문만으로 상대의 논리를 시험합니다.

## EXECUTE

아래 순서로 소크라테스식 대화를 시작해보세요:

**1. 주제 선택** — 본 수업에서 선택한 고전 텍스트(`자료/내선택.md` 참고)의 [보편 논점] 중 하나를 화두로 가져오면 됩니다. 자유 주제도 가능합니다.

**2. 대화 시작** — 선택한 주제에 대해 아래처럼 대화를 시작해보세요:

```
이 주제에 대해 어떻게 생각하시나요?
```

AI(소크라테스)가 질문만으로 대화를 이끌 것입니다. 3~5턴 정도 대화해보세요. 대화가 끝나면 "완료"라고 입력해주세요.

> **팁**: 답변할 때 자기 생각을 솔직하게 말씀해주세요. 정답은 없습니다!


- 어떤 질문이 왜 어려웠는지 공감한다

- 소크라테스식 대화의 가치를 한 문장으로 정리한다
