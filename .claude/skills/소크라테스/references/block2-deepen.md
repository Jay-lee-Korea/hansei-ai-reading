# Block 2: 심화 대화 — 전제 탐색과 반례 제시

## EXPLAIN

🏛️ **대화를 깊게 만드는 3가지 기법**

Block 1에서 소크라테스식 대화를 체험했습니다. 이번에는 대화를 **더 깊게** 만드는 기법 3가지를 배웁니다.

### 기법 1: 전제 질문

상대의 답변에서 **당연하게 여기는 가정**을 찾아 질문합니다.

```
학생: "교육의 목적은 취업이에요."
소크라테스: "왜 그렇다고 생각하시나요? 취업 이전에 교육이 존재했던 시대에는
           교육의 목적이 무엇이었을까요?"
```

### 기법 2: 반례 제시

상대의 주장이 **성립하지 않는 경우**를 질문 형태로 제시합니다.

```
학생: "법을 지키는 것이 항상 옳아요."
소크라테스: "그렇다면 이런 경우는 어떨까요? 인종차별을 합법화했던 법이 있었는데,
           그 법을 지키는 것도 옳은 건가요?"
```

### 기법 3: 종합 질문

대화 전체를 돌아보며 **생각의 변화**를 확인합니다.

```
소크라테스: "우리가 대화를 시작할 때 '정의는 강자의 이익'이라고 하셨는데,
           지금도 같은 생각이신가요? 달라진 부분이 있다면 무엇인가요?"
```

| 기법 | 핵심 질문 패턴 | 목적 |
|------|---------------|------|
| 전제 질문 | "왜 그렇다고 생각하시나요?" | 숨겨진 가정 드러내기 |
| 반례 제시 | "이런 경우는 어떨까요?" | 주장의 한계 발견하기 |
| 종합 질문 | "처음 생각과 지금 생각이 달라진 부분은?" | 사고 발전 확인하기 |

## EXECUTE

이번에는 **5턴 이상의 깊은 대화**를 해봅니다.

**규칙:**
- AI(소크라테스)는 오직 질문으로만 대화합니다
- 학생은 질문에 대해 자유롭게 답변합니다
- 전제 질문, 반례 제시, 종합 질문이 자연스럽게 포함됩니다
- 대화를 끝내고 싶으면 **"그만"**이라고 말씀해주세요

**시작 방법:**
1. Block 1에서 선택한 주제를 이어가거나, 새로운 주제를 정해주세요
2. 자신의 생각을 한 문장으로 말씀해주시면, 소크라테스가 질문을 시작합니다

> **목표**: 대화가 끝난 후, 처음 생각과 달라진 부분이 있는지 돌아보세요.

3. [핵심 질문 3]
