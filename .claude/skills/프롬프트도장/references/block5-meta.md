# Block 5: 메타 프롬프팅

## EXPLAIN

🥋 **사범 수련: AI에게 프롬프트 자체를 설계하게 합니다. 최종 기법입니다.**

### Before (직접 프롬프트 작성)

학생이 직접 고민해서 프롬프트를 쓴다. 시간이 오래 걸리고 빠뜨리는 요소가 있다.

### After (메타 프롬프팅)

```
나는 고전 텍스트에 대한 비판적 에세이를 쓰고 싶어.
이 목적에 가장 적합한 프롬프트를 설계해줘.

프롬프트에 포함해야 할 요소:
- 역할 (페르소나)
- 맥락 (어떤 텍스트인지)
- 구체적 지시 (무엇을 해야 하는지)
- 출력 형식 (어떤 구조로)
- 제약 조건 (하지 말아야 할 것)
```

| 핵심 | 설명 |
|------|------|
| 🔄 메타 수준 | 프롬프트를 직접 쓰는 대신, "프롬프트를 만들어줘"라고 요청 |
| 🧩 요소 지정 | 프롬프트에 포함할 요소를 명시한다 |
| ⚡ 효율성 | 전문적인 프롬프트를 빠르게 얻을 수 있다 |

**메타 프롬프팅**: "프롬프트를 만드는 프롬프트". 지금까지 배운 4가지 기법(컨텍스트, 페르소나, 퓨샷, 체인오브소트)을 AI가 조합하게 한다.

### 지금까지 배운 5가지 기법 관계도

```
메타 프롬프팅 (사범)
  ├── 컨텍스트 주입 (입문) ← 범위 + 연결 + 형식
  ├── 페르소나 (초급) ← 역할 + 행동 + 구조
  ├── 퓨샷 (중급) ← 예시로 패턴 학습
  └── 체인오브소트 (고급) ← 단계적 사고 유도
```

## EXECUTE

아래를 실행해보세요:

**1. 메타 프롬프트 생성** — 자신의 목적에 맞게 요청하세요:

```
나는 [고전 텍스트 이름]에 대한 비판적 에세이를 쓰고 싶어.
이 목적에 가장 적합한 프롬프트를 설계해줘.
프롬프트에 역할, 맥락, 구체적 지시, 출력 형식, 제약 조건을 포함해줘.
```

**2. 생성된 프롬프트 실행** — AI가 만들어준 프롬프트를 그대로 실행해보세요.

**3. 결과 평가** — 직접 쓴 프롬프트와 메타 프롬프트 결과를 비교해보세요.
