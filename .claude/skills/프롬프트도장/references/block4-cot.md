# Block 4: 단계적 사고 (체인오브소트)

## EXPLAIN

🥋 **고급 수련: 복잡한 문제를 단계로 쪼개면, AI가 더 정확하게 답합니다.**

### Before (나쁜 프롬프트)

```
이 논증이 타당한지 판단해줘
```

문제: AI가 한꺼번에 판단하려다 중요한 논리적 결함을 놓친다.

### After (좋은 프롬프트)

```
이 논증을 아래 단계로 검토해줘:
1) 먼저 핵심 주장을 한 문장으로 정리
2) 주장을 뒷받침하는 전제를 나열
3) 각 전제가 참인지 검토
4) 전제에서 결론으로의 논리적 연결이 유효한지 평가
5) 최종 판단과 그 이유를 제시
```

| 핵심 | 설명 |
|------|------|
| 🪜 단계 분해 | 복잡한 사고를 순서 있는 단계로 쪼갠다 |
| 🔍 중간 점검 | 각 단계에서 AI가 자기 사고를 검증한다 |
| 🎯 정확도 향상 | 한꺼번에 답하는 것보다 정확도가 높아진다 |

**체인오브소트(Chain-of-Thought)**: AI에게 "생각의 사슬"을 보여달라고 요청하는 기법. 단계별로 사고하게 하면 복잡한 추론에서 실수가 줄어든다.

## EXECUTE

아래를 순서대로 실행해보세요:

**1. Before 체험** — 본인이 선택한 고전 텍스트(`자료/내선택.md` 자동 사용)를 읽히고:

```
고전텍스트/[파일명]을 읽고 저자의 핵심 논증이 타당한지 판단해줘
```

**2. After 체험** — 같은 질문을 단계적으로:

```
고전텍스트/[파일명]을 읽고 아래 단계로 저자의 논증을 검토해줘:
1) 핵심 주장을 한 문장으로 정리
2) 주장을 뒷받침하는 전제 3가지를 나열
3) 각 전제가 참인지 하나씩 검토
4) 전제에서 결론으로의 논리적 연결을 평가
5) 최종 판단: 이 논증은 타당한가?
```

두 결과를 비교해보세요. 단계적 사고가 어떤 차이를 만드나요?
