# Block 3: 예시 제공 (퓨샷 프롬프팅)

## EXPLAIN

🥋 **중급 수련: 원하는 출력의 예시를 보여주면, AI가 패턴을 따라합니다.**

### Before (나쁜 프롬프트)

```
이 문장을 학술적으로 고쳐줘: "AI가 사람 일자리를 빼앗을 거다"
```

문제: "학술적"이 구체적으로 어떤 수준인지, 어떤 문체인지 AI가 모른다.

### After (좋은 프롬프트)

```
아래 예시처럼 일상 문장을 학술 문장으로 바꿔줘.

예시 1)
일상: "사람들은 돈이 많으면 행복하다고 생각해"
학술: "경제적 풍요와 주관적 행복감 사이의 정적 상관관계는 대중적 통념으로 자리 잡고 있다"

예시 2)
일상: "옛날 사람들은 신을 믿었다"
학술: "전근대 사회에서 종교적 신념 체계는 사회 구성원의 세계관을 규정하는 핵심 기제로 기능했다"

이제 이 문장을 바꿔줘:
"AI가 사람 일자리를 빼앗을 거다"
```

| 핵심 | 설명 |
|------|------|
| 🔖 예시 제공 | 원하는 입출력 쌍을 2~3개 보여준다 |
| 📐 패턴 학습 | AI가 예시의 변환 패턴을 파악한다 |
| 🎯 일관된 품질 | 예시 수준에 맞춰 출력한다 |

**퓨샷(Few-shot) 프롬프팅**: "몇 가지(few) 사례(shot)를 보여주는" 기법. 설명 대신 예시로 가르친다.

## EXECUTE

아래를 순서대로 실행해보세요:

**1. Before 체험:**

```
이 문장을 학술적으로 고쳐줘: "AI가 사람 일자리를 빼앗을 거다"
```

**2. After 체험** — 위 EXPLAIN의 예시 포함 프롬프트를 그대로 입력하세요.

**3. 나만의 퓨샷** — 다른 변환 작업에 예시를 만들어보세요:
- 예: 한국어 → 영어 학술 초록 변환
- 예: 장문 → 트윗 길이 요약
