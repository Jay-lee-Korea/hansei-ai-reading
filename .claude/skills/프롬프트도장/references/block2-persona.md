# Block 2: 역할 부여 (페르소나 프롬프팅)

## EXPLAIN

🥋 **초급 수련: AI에게 역할을 주면, 전문가처럼 답합니다.**

### Before (나쁜 프롬프트)

```
이 텍스트의 문제점을 찾아줘
```

문제: AI가 누구의 관점에서, 어떤 수준으로 검토해야 하는지 모른다.

### After (좋은 프롬프트)

```
너는 20년차 철학과 교수야.
이 텍스트를 석사 논문 심사하듯 비판적으로 검토해줘.
논리적 약점 3가지를 지적하고 각각 개선안을 제시해줘.
```

| 장치 | 역할 |
|------|------|
| 🎭 역할 지정 | "20년차 철학과 교수" → 전문성 수준 설정 |
| 🎯 행동 지정 | "석사 논문 심사하듯 비판적으로" → 태도 설정 |
| 📋 출력 구조 | "약점 3가지 + 개선안" → 구체적 형식 |

**페르소나 프롬프팅**: 역할을 지정하면 AI의 전문성 수준, 관점, 어투가 모두 달라진다.

### 활용 예시

- "너는 소크라테스야. 질문으로만 대화해줘" → 산파술 체험
- "너는 마케팅 전문가야. 이 제품의 약점을 찾아줘" → 비즈니스 관점
- "너는 초등학생이야. 이 개념을 네 말로 설명해봐" → 이해도 확인

## EXECUTE

아래를 순서대로 실행해보세요:

**1. Before 체험** — 고전텍스트 폴더의 파일에 대해:

```
동굴의비유.txt를 읽고 문제점을 찾아줘
```

**2. After 체험** — 같은 파일에 대해:

```
너는 20년차 철학과 교수야.
동굴의비유.txt를 읽고 석사 논문 심사하듯 비판적으로 검토해줘.
논리적 약점 3가지를 지적하고 각각 개선안을 제시해줘.
```

**3. 나만의 페르소나** — 다른 역할을 직접 만들어서 시도해보세요:
- 예: "너는 마르크스야. 이 텍스트를 계급론 관점에서 비평해줘"
