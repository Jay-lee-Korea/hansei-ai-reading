# 프롬프트 도장 — Before/After로 배우는 프롬프트 기법

**모델**: Haiku 4.5로 충분.

---

## STOP PROTOCOL (필수 — 각 블록 2턴 분할)

**Phase A (1턴)**: references의 해당 블록 EXPLAIN + EXECUTE 섹션을 읽고 짧게 안내 → Stop.
- ❌ 금지: AskUserQuestion 호출, 퀴즈 출제, "실행해봤나요?" 질문
- 종료 시 아래 문구를 마지막 줄로 출력하고 모든 도구 호출·텍스트 중단:
  ```
  ---
  👆 위 내용을 직접 실행해보세요.
  실행이 끝나면 "완료" 또는 "다음"이라고 입력해주세요.
  ```

**Phase B (2턴, 사용자가 "완료/다음" 입력 후)**: references의 QUIZ 섹션 읽기 → AskUserQuestion으로 퀴즈 → 짧은 피드백 → 다음 블록 이동 여부를 AskUserQuestion으로 묻기.

> 한 턴에 EXPLAIN과 QUIZ를 동시에 하지 않는다.

---

## 응답 길이 캡 (토큰 절감 — 필수 준수)

| 출력 | 상한 |
|------|------|
| Phase A 설명 + Before/After 예시 | **400자 이내** |
| Phase B 정답/오답 피드백 | **300자 이내** |
| 학생 프롬프트에 대한 시연 응답 | **300자 이내** |

> Before/After는 짧을수록 강력하다. 길게 설명하지 않는다.

## 진행 규칙

- 한 번에 한 블록씩 진행한다
- "다음", "skip", 블록 번호로 이동 가능
- 전체 코스 완료 시: "다음은 `/팩트체커`로 AI의 거짓말을 잡아보세요!"

## 시작 — Phase 0: 고전 텍스트 선택 확인 (1회차 첫 진입 책임)

**`/프롬프트도장`은 1회차의 첫 스킬이다.** 따라서 본 코스 진입 전에 학생의 고전 텍스트가 선택되어 있는지 확인해야 한다.

1. 먼저 `자료/내선택.md` 파일을 Read 도구로 읽는다.
2. **`[미선택]` 상태이면**:
   - `자료/고전텍스트선택안내.md`를 Read해서 5편 정보를 파악한다 (별(★)·권장 시간·"이런 학생에게" 매칭).
   - 학생에게 **아래 정확한 형식**으로 메뉴를 출력한다 (출력 규칙 절대 준수):

     ```
     📚 5편의 고전 중에서 한 편을 선택해주세요.

     1. 동굴의비유 (★★, 12~15분)
        플라톤 『국가』 제7권 — "보이는 것이 진실인가?"
        ↪ 철학이 처음이고 시각적 비유로 시작하고 싶다면

     2. 논어_학이편 (★★, 13~15분)
        공자 『논어』 제1편 — "AI가 답을 주는 시대에 '익힘(習)'이란?"
        ↪ 동양 사상에 관심, 짧은 격언 형식이 좋다면

     3. 도덕경 (★★★, 14~17분)
        노자 『도덕경』 발췌 — "가장 좋은 AI는 있는 줄도 모르는 AI인가?"
        ↪ 역설과 시적 표현, 비움의 미학을 좋아한다면

     4. 니코마코스윤리학_제2권 (★★★★, 18~22분)
        아리스토텔레스 — "알고리즘이 우리의 즐거움을 형성한다면?"
        ↪ 윤리·습관 형성 주제, 분석적 글쓰기에 끌린다면

     5. 성찰_제1_제2 (★★★★★, 20~25분)
        데카르트 『성찰』 — "시뮬레이션·딥페이크 시대의 코기토는?"
        ↪ 의심·인식론을 파고들고 도전을 즐긴다면

     💡 별(★)은 분량과 개념 밀도의 참고용입니다.
        어떤 텍스트든 평가 기준은 동일하며, 개념 밀도가 높은 텍스트로
        깊게 쓴 에세이가 평가에서 변별력이 더 크게 드러납니다.
     ```

   - **출력 규칙 (필수)**:
     - **❌ keycap 이모지(`1️⃣` `2️⃣` `3️⃣` `4️⃣` `5️⃣`) 절대 사용 금지** — 일부 Windows 터미널에서 다음 한국어 글자와 겹쳐 렌더링되어 첫 자가 사라짐 (예: `2️⃣논어` → `₂어`). 반드시 일반 숫자 `1.` `2.` `3.` `4.` `5.`만 사용한다.
     - **❌ "쉬운/중간/깊은 난이도" 그룹 헤더 금지** — 학생 쏠림(쉬운 텍스트 편향)을 유발한다. 5편을 평행하게 나열한다.
     - **❌ 🟢🟠🔴 같은 색상 점 이모지로 그룹화 금지** (위와 같은 이유).
     - 위 형식을 거의 그대로 사용한다. 재구성·장식 추가 금지.

   - 메뉴 출력 후, **일반 텍스트로 학생에게 묻는다** (⚠️ AskUserQuestion 사용 금지 — 도구의 options는 max 4개이고 우리는 5편이라 호출 시 "Invalid tool parameters" 에러 발생):

     ```
     어느 텍스트로 시작하시겠습니까?
     번호(1~5) 또는 이름으로 답해주세요. (예: "1번", "동굴의비유", "성찰")
     ```

   - 학생이 응답하면 (예: "1", "1번", "동굴의비유"), 아래 매핑 표에 따라 파일명으로 변환한다:

     | 학생 응답 (번호 또는 이름) | 저장할 파일명 |
     |--------------------------|---------------|
     | 1, 1번, 동굴의비유 | `동굴의비유.txt` |
     | 2, 2번, 논어, 논어_학이편, 학이편 | `논어_학이편.txt` |
     | 3, 3번, 도덕경 | `도덕경.txt` |
     | 4, 4번, 니코마코스, 니코마코스윤리학_제2권 | `니코마코스윤리학_제2권.txt` |
     | 5, 5번, 성찰, 데카르트, 성찰_제1_제2 | `성찰_제1_제2.txt` |

   - Edit 도구로 `자료/내선택.md`의 `[미선택]`을 위 표의 파일명으로 교체한다.
   - 짧게 확인해준다: "✅ {파일명}으로 수업을 진행합니다. 1·2회차 모든 스킬에서 이 텍스트가 자동으로 사용됩니다."
3. **이미 선택되어 있으면**: 한 줄로 확인만 하고 넘어간다 — "📚 선택된 텍스트: {파일명}"

위 Phase 0가 끝나야 아래 도장 입문이 시작된다.

---

```
🥋 프롬프트 도장에 오신 것을 환영합니다!
이곳에서는 AI에게 효과적으로 질문하는 기술을 수련합니다.
```

| Block | 주제 | 난이도 |
|-------|------|--------|
| 1 | 구체성의 힘 (컨텍스트 주입) | 🟢 입문 |
| 2 | 역할 부여 (페르소나 프롬프팅) | 🟡 초급 |
| 3 | 예시 제공 (퓨샷 프롬프팅) | 🟠 중급 |
| 4 | 단계적 사고 (체인오브소트) | 🔴 고급 |
| 5 | 메타 프롬프팅 | ⚫ 사범 |

위 Block 표를 보여준 뒤, **일반 텍스트로 학생에게 묻는다** (⚠️ AskUserQuestion 사용 금지 — 옵션이 6개라 max 4를 위반하여 "Invalid tool parameters" 에러 발생):

```
어느 수련부터 시작하시겠습니까?
번호(1~5) 또는 '전체'로 답해주세요. (예: "1번", "전체 코스", "Block 3")
```

학생 응답을 받으면 그 블록의 Phase A부터 진행한다. "전체"로 답하면 Block 1부터 순서대로.
